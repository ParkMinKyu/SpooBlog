# 이슈 작성 방법
- 이슈 제목은 자신의 이름을 입력하고 내용은 자신이 맡은 부분의 기능을 쪼개어 아래의 형식으로 작성한다.

>[ ] 기능1

>[ ] 기능2

- 개발이 완료된 기능은 [] 안에 x를 추가한다.

> [ x ] 기능1

> [ ] 기능2

- 정상적으로 작성한 이슈는 [#2](https://github.com/SpoonCode/SpooBlog/issues/2) 처럼 보인다.
